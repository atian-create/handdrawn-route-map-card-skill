# Sample Handdrawn Route Map Card Brief

## Input

```text
做一张图文发布工作流地图。步骤：收集灵感、判断选题、写大纲、生成配图、发布检查、数据复盘。风格：奶油纸底水彩路线图。
```

## Output Shape

Return a title, subtitle, 6 route nodes, side collage, mini deliverables box, bottom note, generation brief, quality checklist, and optional caption.

## Boundary Note

This sample uses user-provided material and public-safe judgment only. It does
not require platform login, automatic publishing, private analytics connectors,
or copying another creator's content.
